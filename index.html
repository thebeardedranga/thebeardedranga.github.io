ScriptableObject Architect

The definitive spreadsheet-style database editor for Unity 6.

[ Image Placeholder: ScriptableObject_Architect_Preview.png ]

Manage your project's data at scale. ScriptableObject Architect uses Unity 6's native UI Toolkit to provide a high-performance, multi-column interface for mass-editing data assets without the overhead of heavy third-party frameworks.

  • Multi-Column List View: View and edit any ScriptableObject field in a familiar spreadsheet layout.
  • Reflective Discovery: Automatically finds all assets of a selected type within your project.
  • Unity 6 Optimized: Built with native C# and UI Toolkit—no Odin or IMGUI bloat.
  • Asset Store Ready: Compiled core logic for security and minimal domain-reload impact.
